# Java 25

## Overview
Java 25 introduces new features and improvements to the Java platform.
This repository contains Examples how to use these new features and improvements

## Getting Started
- Install Java 25 Development Kit (JDK)
- Set up your development environment
- Navigate to the corresponding `src` directory
- Invoke `Java XYZ.java` to execute the example
- Some examples need additional command line parameters. See javadoc in the *.java files.
